# Lunagraph documentation project

## About this project

- Documentation site for Lunagraph, built on [Mintlify](https://mintlify.com)
- Pages are MDX files with YAML frontmatter
- Configuration lives in `docs.json`
- Run `mint dev` to preview locally

## Terminology

- "Canvas" — the visual design surface in Lunagraph
- "Round-trip editing" — changes in the canvas sync to code and vice versa

## Style preferences

- Use active voice and second person ("you")
- Keep sentences concise
- Use sentence case for headings
